# 💬 Real-Time Chat Application

A modern, real-time chat application built with Spring Boot and WebSocket technology. This application allows users to register, login, and communicate with each other in real-time with message persistence.

## ✨ Features

### Core Features
- ✅ **User Authentication**: Secure registration and login with BCrypt password hashing
- ✅ **Real-time Messaging**: Instant message delivery using WebSocket
- ✅ **Private Conversations**: One-on-one private chat between users
- ✅ **Message History**: Persistent storage of all messages in MySQL database
- ✅ **Online Status**: Real-time user online/offline status
- ✅ **User Search**: Search and find other users
- ✅ **Session Management**: Secure HTTP session-based authentication
- ✅ **Responsive Design**: Works seamlessly on desktop and mobile devices

### Technical Features
- 🔐 **Security**: Spring Security with BCrypt encryption
- 📡 **WebSocket**: Real-time bidirectional communication
- 💾 **Persistence**: JPA/Hibernate with MySQL database
- 🎨 **Modern UI**: Clean, intuitive interface with CSS3
- 📱 **Mobile Responsive**: Adapts to different screen sizes
- 🔄 **Auto-scroll**: Automatic scrolling to latest messages
- ⏱️ **Timestamps**: Accurate message timestamps
